# 📖 Welcome to DevDocs

Welcome to your personal DevOps documentation tool! This is a simple, file-based documentation system where every document is a Markdown (`.md`) file.

## 🚀 Quick Start

### Creating Documents

1. Click **📄 New File** in the sidebar to create a document
2. Click **📁 New Folder** to create a category
3. Use the **+** buttons on any folder to add files inside it

### Editing

- Click any document in the sidebar to view it
- Press **✏️ Edit** or `Ctrl+E` to enter edit mode
- Write Markdown in the left pane, see live preview on the right
- Press **💾 Save** or `Ctrl+S` to save

### Adding Screenshots

You can add images in three ways:

1. **Drag & Drop** — Drag an image file into the editor
2. **Paste** — Copy a screenshot and press `Ctrl+V` in the editor
3. **Upload Button** — Click the 🖼️ button in the toolbar

The image will be uploaded automatically and the Markdown tag inserted at your cursor!

## ⌨️ Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+E` | Edit current document |
| `Ctrl+S` | Save document |
| `Escape` | Cancel editing |
| `Ctrl+K` | Focus search |

## 📁 Folder Structure

Organize your docs by creating folders for each topic:

```
docs/
├── docker/
│   ├── basics.md
│   └── networking.md
├── kubernetes/
│   ├── pods.md
│   └── services.md
└── ci-cd/
    ├── github-actions.md
    └── jenkins.md
```

Each folder becomes a **category** in the sidebar.

## 🔍 Search

Use the search bar at the top of the sidebar to find any text across all your documents. Results will show matching snippets for context.

---

> **Tip:** Right-click any file or folder in the sidebar for more options like rename and delete!
