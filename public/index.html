📘 DevDocs Manual

📁 File & Folder Management

Action How
Create file Click 📄 New File in sidebar or right-click a folder → New File Here
Create folder Click 📁 New Folder in sidebar or right-click a folder → New Folder Here
Rename Right-click file/folder → ✏️ Rename
Delete Click 🗑️ on hover, or right-click → Delete (moves to Trash)
Restore Open Trash panel → click ↩ Restore

✍️ Editing Documents

Action How
Edit a doc Open any file → click ✏️ Edit or press Ctrl+E
Save changes Click 💾 Save or press Ctrl+S
Cancel editing Click ✖ Cancel or press Esc
New paragraph Press Enter twice for a new paragraph
Tab indent Press Tab to insert 2 spaces

🖼️ Images

Action How
Upload via button While editing, click 🖼️ Image in toolbar
Drag & drop Drag an image file onto the editor area
Paste from clipboard Copy an image and press Ctrl+V in the editor
Image lifecycle Images are automatically deleted when their parent file/folder is deleted

⌨️ Keyboard Shortcuts

Shortcut Action
Ctrl+E Enter edit mode
Ctrl+S Save document
Esc Cancel editing / close menus
Ctrl+K Focus search bar
Tab Insert 2 spaces (in editor)

🗑️ Trash & Recovery

  • Deleted files and folders are moved to Trash, not permanently deleted
  • Images referenced in deleted files are also moved to trash
  • Click 🗑️ Trash in the sidebar to view/restore/permanently delete items
  • Use Empty Trash to permanently remove all trashed items

💡 Code Highlighting

Use fenced code blocks with a language tag for automatic syntax highlighting:

```javascript
const app = express();
```

```bash
docker build -t myapp .
```

```yaml
version: '3.8'
services:
  app:
    build: .
```

Supported: javascript, python, bash, yaml, dockerfile, json, html, css, sql, go, rust, and 180+ more.
